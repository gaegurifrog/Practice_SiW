## Important vocabulary for software programming and development

#### Verbs

* get, set, reset, find
* is, has
* convert, replace, format
* insert, append, add
* remove, delete
* close, open
* make, copy, create
* read, write

#### Nouns

* constant, variable, value
* sequence, iterable, element
* function, argument, parameter, flag, operator
* class, object, instance, method, attribute
* loop, iteration, index
* dictionary, key

#### Adjectives

* default, optional
* mutable, immutable

#### Vocab for Collaboration

* open source
* repository
* license
* issue
* bug
* feature request
* branch
* pull request
* fork
* clone
